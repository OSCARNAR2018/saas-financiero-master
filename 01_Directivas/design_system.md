# DESIGN SYSTEM: VISIONARIOS BOLSA (PREMIUM DARK MODE)

## Paleta de Colores
- **Fondo Global (Black Hole)**: `#050505`
- **Fondo Tarjeta (Deep Space)**: `#0F0F0F` con opacidad del 70% (Glassmorphism)
- **Borde Primario (Slate Glow)**: `#1F1F1F`
- **Acento Primario (Electric Blue)**: `#00A3FF`
- **Éxito (Neon Green)**: `#00FF94`
- **Peligro (Radiant Red)**: `#FF3B3B`
- **Texto Primario**: `#FFFFFF`
- **Texto Secundario**: `#A0A0A0`

## Efectos de Interfaz
- **Glassmorphism**: `backdrop-blur-md`, `bg-opacity-70`, `border-opacity-20`
- **Sombras**: Sombras sutiles de color (Glow) para elementos activos.
- **Interacción (Hover)**: Escalamiento `scale-101` y brillo del borde.

## Tipografía
- **Main Font**: 'Inter' o 'Outfit' (Sans-serif moderna).
- **Weights**: 400 (Base), 600 (Semibold), 700 (Bold).

## Componentes UI
- **Badges**: Bordes redondeados `rounded-full`, padding horizontal generoso.
- **Tarjetas**: `rounded-2xl`, bordes finos.

# SaaS Financiero Master

Plataforma integral de análisis financiero unificado para Big Caps, Small Caps y Opciones.

## Estructura del Proyecto

- `01_Directivas`: Objetivos de negocio y estrategias de inversión.
- `02_Orquestacion`: Lógica de selección de activos y coordinación de agentes.
- `03_Ejecucion`: Scripts de procesamiento, automatización de n8n y configuración de base de datos.
- `frontend`: Dashboard moderno construido con Next.js, React y Tailwind CSS, integrado con Supabase.

## Requisitos

- Node.js 18+
- Python 3.9+
- Cuenta en Supabase
- Instancia de n8n (opcional para automatización)

## Configuración Local

1. Clona el repositorio.
2. Configura las variables de entorno en un archivo `.env` basado en la documentación proporcionada.
3. Instala dependencias del frontend: `cd frontend && npm install`.
4. Ejecuta el entorno de desarrollo: `npm run dev`.
